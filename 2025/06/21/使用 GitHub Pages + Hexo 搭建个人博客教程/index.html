配置 GitHub

  1. 配置 GitHub SSH 密钥

    • a. 生成 SSH Key：

    • b. 添加 SSH Key 到 GitHub：

      1. 登录 GitHub，点击右上角头像 -> Settings。
      2. 在左侧边栏选择 SSH and GPG keys。
      3. 点击 New SSH key 按钮。
      4. 在 Title 字段输入一个易于识别的名称（如 My PC Key）。
      5. 打开本地生成的 id_rsa.pub 文件（通常在 ~/.ssh/ 目录下），复制其全部内容。
      6. 将内容粘贴到 Key 文本框中。
      7. 点击 Add SSH key。

    • c. 测试 SSH 连接：

    • d. 配置 Git 全局用户信息：

  2. 更换主题 (以 Stellar / Next 为例)

    • 方式一：手动下载 (如 Stellar)

      1. 访问主题的 GitHub 仓库 (如：https://github.com/xaoxuu/hexo-theme-stellar)。
      2. 下载 ZIP 压缩包。
      3. 解压到博客根目录下的 themes/ 文件夹中，确保主题文件夹名为 stellar。

    提示： Hexo 官网的 主题页面 提供了丰富的主题选择。

部署到 GitHub Pages

  1. 创建 GitHub Pages 仓库

    • 在 GitHub 上创建一个新的仓库。
    • 仓库名必须严格遵守格式： 你的GitHub用户名.github.io (例如：huanyuehjj.github.io)。这个仓库名将直接作为你博客的访问地址。

  3. 部署博客

    • 确保已安装 hexo-deployer-git 插件 (见步骤 4)。

    • 执行部署命令：

    • 首次部署可能需要输入 GitHub 账户密码或授权。

    • 部署完成后，稍等几分钟（GitHub Pages 需要构建），在浏览器访问 https://你的GitHub用户名.github.io (例如：https://huanyuehjj.github.io) 即可看到你的在线博客。

恭喜！

至此，你的个人博客已成功部署到 GitHub Pages！接下来，你可以专注于编写博客内容，并深入探索 Hexo 的配置选项（如站点信息、主题样式、插件等）来个性化你的博客。
